A simple timer

Install yeoman and associated dependencies.

    Npm install -g yo grunt-cli gulp

Pull down the source

Install bower and Npm things

    Npm install

    Bower install

Serve up the app
    Grunt serve

Run the tests 
    Grunt test

Build 
    Grunt
